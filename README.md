# Hisab Kitab 💰

HisabKitab is a daily expense tracking application that helps users to manage their income and expense efficiently.  
  
HIsabKitab stores financial data locally on your device using SQLite and provide insights, but also syncs into firebase after connecting to the internet.


## 🚀 Features
- Add and manage income and expense transactions
- Dashboard showing balance, income, and expenses
- Recent transaction history
- Dynamic sorting of transactions by date
- Analytics page with graphical insights

## 🛠 Tech Used
- Java
- Android Studio
- Firebase
- SQLite
- XML
- MPAndroidChart

## 🧠 Algorithms Used  

- Aggregation Algorithm 
- Trend Analysis
- Moving Average Prediction

## 📈 Data Visualization

The application uses MPAndroidChart to visualize financial data.

Charts used:
- Pie Chart – Income and expense distribution
- Bar Chart – Spending trends
- Line Chart – Transaction activity over time
