# Hisab Kitab 💰

An Android Expense Tracker App built using Java.

## Features
- Add income
- Add expenses
- Track balance
- Simple UI

## Tech Used
- Java
- Android Studio
- Firebase
- SQLite

## Algorithms Used 

- Aggregation Algorithm 
- Trend Analysis
- Moving Average Prediction
